# 🌿 Landing Page – Dara Bragança Massoterapeuta

Uma landing page moderna, responsiva e otimizada para profissionais da área de bem-estar e massoterapia. Desenvolvida com **HTML, CSS e JavaScript puros**, focada em conversão, experiência do usuário e design clean.

---

## ✨ Funcionalidades

- ✅ **100% Responsivo** – Adapta-se perfeitamente a mobile, tablet e desktop
- 📱 **Integração com WhatsApp** – Botão flutuante e CTAs com mensagem pré-preenchida
- 🎠 **Carrossel Interativo** – Autoplay, navegação por setas, indicadores e suporte a swipe
- 🎨 **CSS Variables** – Paleta de cores centralizada para fácil personalização
- ⚡ **Zero Dependências** – Apenas HTML5, CSS3 e Vanilla JavaScript
- 🎭 **Animações Suaves** – Scroll behavior, fade-ins e hover effects elegantes
- 🔍 **SEO Básico** – Tags semânticas, meta viewport e estrutura acessível

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia       | Arquivo        | Descrição                              |
|------------------|----------------|----------------------------------------|
| HTML5            | `index.html`   | Estrutura semântica da página          |
| CSS3             | `style.css`    | Estilização, Grid, Flexbox, Animações  |
| JavaScript       | `script.js`    | Carrossel e interações                 |
